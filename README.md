# meetup_pbi

Material do meetup **"Do prompt ao dashboard: um novo fluxo de trabalho com IA e Power BI"** — skills de Power BI / Fabric para Claude Code + apresentação da aula.

## Conteúdo

- [`apresentacao.html`](apresentacao.html) — slides da apresentação (abra no navegador; navegue com ← → ou clique)
- [`skills/`](skills/) — skills de Power BI / Fabric para Claude Code

Origem das skills: [microsoft/skills-for-fabric](https://github.com/microsoft/skills-for-fabric) (MIT License, veja [LICENSE](LICENSE)).

## Skills incluídas

- `skills/powerbi-report-authoring`
- `skills/powerbi-report-design`
- `skills/powerbi-report-management`
- `skills/powerbi-report-planning`
- `skills/semantic-model-authoring`

## Instalação

```bash
git clone https://github.com/soutes/meetup_pbi.git
```

Copie as pastas desejadas de `skills/` para:

- `%USERPROFILE%\.claude\skills\` (skills globais), ou
- `.claude\skills\` dentro do projeto (skills locais)

Reinicie o Claude Code / OpenClaude para carregar.
