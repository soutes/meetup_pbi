# skills

Power BI / Fabric skills for Claude Code.

Origem: [microsoft/skills-for-fabric](https://github.com/microsoft/skills-for-fabric) (MIT License, veja [LICENSE](LICENSE)).

## Skills incluídas

- `powerbi-report-authoring`
- `powerbi-report-design`
- `powerbi-report-management`
- `powerbi-report-planning`
- `semantic-model-authoring`

## Instalação

```bash
git clone https://github.com/soutes/skills.git
```

Copie as pastas de skills desejadas para:

- `%USERPROFILE%\.claude\skills\` (skills globais), ou
- `.claude\skills\` dentro do projeto (skills locais)

Reinicie o Claude Code / OpenClaude para carregar.
